# Diagnosis or Delusion: Revenant Investigation Game

An interactive terminal game exploring how folklore emerges from misunderstood natural phenomena.

## How to Play

1. Download this repository (click "Code" then "Download ZIP")
2. Extract the zip file
3. Open PowerShell or Command Prompt
4. Navigate to the folder: `cd vampire-game`
5. Install dependencies: `npm install`
6. Run the game: `node index.js`
7. Use arrow keys to select investigation paths and diagnoses

## About

This game investigates three historical 18th-century revenant cases:
- **Arnod Paole (1727, Serbia)**: Soldier allegedly returning from death
- **Friedrich Von Seckendorf (1754, Austria)**: Nobleman's remarkably preserved body
- **Peter Plogojowitz (1725, Serbia)**: Most documented case with adipocere formation
